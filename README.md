# Instructions on Installing morecmath
To get morecmath running on your system, just install the files using
__git clone -b master --single-branch https://github.com/WardenOfTheCove/wardenofthecove.github.io.git__
Then follow these steps:
1. Go into Calculator folder
2. Go into mathFuncs/build directory
3. Run the __make__ command
4. Run the __sudo make install__ command
5. Use the library as <morecmath/funcs.hpp>
Have fun!
