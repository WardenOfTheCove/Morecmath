# MoreCMath
Morecmath is a library for people who want just beyond the capabilities of the standard C++ math library, cmath.
It gives useful features, such as a factorial calulator, cross product calculator, dot product calculator, as well as a
cartesian to spherical coordinate converter.
__I WOULD LIKE TO EMPHASIZE THAT THIS PROJECT DID NOT USE AI__
## Installation
To install the library, just download the morecmath.tar.gz file from the releases and open it.
Run the install script in the mathFuncs folder. The library will be installed and ready for use if you have CMake or Conan.
Currently, I am looking into getting this published through Conan-Center
